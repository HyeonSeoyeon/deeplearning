AI(인공지능)란?
: 데이터의 특성(속성, feature)을 파악하고 활용하는 기술
사람이 직접 규칙을 정하지 않아도, 데이터를 보고 패턴을 찾아 자동으로 문제를 해결하는 기술.

지능(Intelligence)이란?
: 생각하고 판단하는 능력
인간의 두뇌처럼 정보를 이해하고, 결정을 내리는 것을 의미.
-> AI는 이런 지능을 모방하려는 기술임.

classification과 regression의 주된 차이점
: 분류(이산적인 값을 맞추는 것), 회귀(연속적인 값을 맞추는 것)

          Classification (분류)      / 	Regression (회귀)
목적	카테고리(이산적 값) 맞추기   /  숫자(연속적 값) 예측하기
예시	"이 꽃은 어떤 품종인가?"   /	"이 집의 가격은 얼마일까?"
출력값	정해진 클래스(0, 1, 2...)	  /  실수 값(1.5, 2.73...)
사용 모델	의사결정나무, 랜덤포레스트, SVM	/   선형회귀, 랜덤포레스트 회귀

전통적인 프로그램과 ai 프로그램의 차이
: 입력 -> 규칙 -> 출력(정답)   /  입력+출력(정답) -> 규칙 도출-> 이 규칙을 가지고 새로운 데이터를 입력했을 때 정답 출력 이때 규칙 도출하는 과정을 학습이라 한다 (반면 전통적인 프로그래밍은 rule을 프로그래머가 입력)

            전통적인 프로그래밍	AI 프로그램
방식	   사람이 규칙을 직접 입력	데이터에서 규칙을 학습
입력	   입력 → 규칙 → 출력	        입력 + 정답 → 학습 → 규칙 생성
예시	   if-else 조건문, 알고리즘	머신러닝, 딥러닝 모델

쉽게 말하면?
전통적인 프로그래밍: 규칙(Rule)을 사람이 직접 짬 → "이런 상황이면 이렇게 해라!"
AI 프로그램: 데이터를 보고 스스로 규칙을 찾음 → "어떤 규칙이 제일 적절한지 배워서 결정해!"

붓꽃 데이터 (Iris Dataset)에서 X, y 설정
입력(X): 꽃잎 길이, 너비, 꽃받침 길이, 너비
출력(y): 꽃의 품종(species)
X = df.drop(columns=['species'])  # 특성 데이터 (입력값)
y = df['species']  # 타겟 데이터 (정답)

붓꽃데이터에서 species는 y로 출력, 나머지는 x
그리고 모델을 학습시킬 때 데이터를 훈련용(train)과 테스트용(test)으로 나눠야 함. ==> x_train, y_train, x_test, y_test
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)

test 데이터: 모델이 패턴을 익히는 데이터
train 데이터: 모델이 얼마나 잘 맞추는지 검증하는 데이터
-> 모델이 Test 데이터를 잘 맞추면, 실제 상황에서도 잘 작동할 확률이 높다.
train 데이터와 test 데이터의 차이점을 봐야한다 -> 정확성 확인

인공지능>머신러닝>딥러닝 (모두 포함관계)
⬇ AI의 하위 개념
머신러닝(ML)
: AI 중에서 데이터를 기반으로 학습하는 방식.
사람이 직접 규칙을 짜는 대신, 데이터에서 패턴을 찾아 학습.
=> 특징을 먼저 추출하고 모델에 넣는다.
input -> feature extraction -> classification -> output
⬇ 머신러닝의 하위 개념
딥러닝(DL)
: 머신러닝 중에서도 신경망(Neural Networks)을 활용한 모델.
특징을 사람이 설계하지 않아도 됨 → 자동으로 특징(feature)을 추출
=>raw 데이터를 넣고 모델에서 분류
input -> classification -> output

지도학습(Supervised Learning)
: 입력(X)과 정답(y)이 둘 다 주어진 상태에서 학습하는 방법
즉, 이미 답이 있는 데이터를 가지고 모델을 학습시키는 방식.
모델은 입력 데이터(X) 를 보고, 정답(y) 을 예측하도록 훈련.
수업에서 배운 DT, RF, SVM, LR 같은 모델이 대표적인 지도학습 방법.
ex) 붓꽃 분류 (Iris Classification) → 꽃잎 길이, 너비(입력) → 품종(정답)

반지도학습(Semi-Supervised Learning)
: 입력(X) 데이터는 많은데, 정답(y)이 일부만 있는 경우.
지도학습과 비지도학습(정답 없이 패턴을 찾는 방식)의 중간 형태!
ex) 인터넷에서 수집한 뉴스 기사 분류 : 뉴스 기사(입력)는 많지만, 라벨(정치, 경제, 스포츠 등)은 일부만 있음.

강화학습(Reinforcement Learning, RL)
: 보상을 주면서 스스로 학습하는 방식
사람처럼 시행착오를 거치면서 점점 더 좋은 선택을 하도록 학습.
즉, 행동을 하면 보상을 받고, 그 보상을 최대화하도록 배우는 것!
ex) 파고(AlphaGo) : 바둑을 두면서 이기면 보상 +1, 지면 -1 → 점점 더 잘 둠!
